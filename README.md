# Clojure Puzzle

Computer puzzles for studying Clojure programming

## Installation

    $ git clone https://github.com/zerobase/clojure-puzzle.git

## Requirements

You need [Leiningen](http://leiningen.org/).

## Usage

To run a puzzle:

    $ lein run -m puzzle.komachi
